# Team Onboarding

This folder demonstrates how I structure onboarding for new data engineers so they
can become productive quickly while absorbing platform standards and context.

## Components

- `01_onboarding_guide.md`: Narrative guide for new joiners.
- `generate_onboarding_checklist.py`: Script to generate a personalized onboarding 
  check list for a new engineer based on role and seniority.
