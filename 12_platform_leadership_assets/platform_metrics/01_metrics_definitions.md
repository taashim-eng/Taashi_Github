# Metrics That Matter

## Platform Reliability

- Pipeline Success Rate (%)
- Mean Time To Detect (MTTD)
- Mean Time To Resolve (MTTR)
- Number of Incidents by Severity (P0, P1, P2)

## Data Quality

- Number of failed data quality checks per week
- Percentage of critical datasets covered by DQ tests
- Time to resolve data quality issues

## Performance & Cost

- Average query runtime for key dashboards
- Cost per query / per pipeline / per TB processed
- Cluster utilization and auto-scaling effectiveness

## Team Productivity

- Lead time from idea to production for data features
- PR throughput and review times
- Onboarding time to first meaningful contribution
